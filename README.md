# NOBOT platform [![version](https://img.shields.io/badge/version-0.0.1alpha-blue.svg)](https://semver.org)

## Description
Engine for creation bots. Language `BotScript` and translator for it.

## Requierements
* OS: Linux, MacOS

## How to work


## Installing
1. `make install`
2. run `nobot` and pass args if needed

## Uninstalling
1. `make uninstall`
